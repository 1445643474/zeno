安装方法

  • 从官方网站下载（适合国内用户）
  • 从 GitHub 下载（适合国外用户）
  • 首先根据您的平台和硬件下载对应的压缩包文件。例如 Windows，有英伟达显卡的用户，可以下载 zeno-****.*.*-x86_64-windows-cuda.zip 这个文件；如果没有英伟达显卡，则下载 zeno-****.*.*-x86_64-windows-cpu.zip。
  • Windows: 下载后，把 zip 文件解压到一个目录，然后双击其中的 000_start.bat 这个文件，即可启动我们的程序。此外您也可以找到 zenoedit.exe 直接双击运行，或者将其“发送到桌面快捷方式”。
  • Linux: 我们发布的二进制版支持 Ubuntu 18.04 以上的系统 (glibcxx >= 3.4.26, glibc >= 2.14)，否则请从源码构建。把下载的 tar.gz 文件解压到一个目录，然后在那个目录中运行命令 bash 000_start.sh 即可启动我们的程序。此外您也可以直接运行其中的 usr/bin/zenoedit。如果遇到没有执行权限的问题可以试试看 chmod +x usr/bin/*。

问题反馈

如果您遇到了 BUG，或是使用过程中有任何不懂的问题，可以通过发送邮件到 pengyb@zenustech.com 来联系我们帮助您解决, 感谢您的支持！

在反馈的信件中，请尽可能多地附上更多的信息，例如：软件界面截图，控制台报错信息，你使用的显卡型号，或是直接将 zsg 文件作为附件发送，这能帮助我们更快地帮你解决问题。

推荐配置

(这是流畅运行 Zeno 获得最佳体验所需的配置)

  • Windows 10 及以上
  • ... 或 Ubuntu 20.04 及以上
  • 至少 4 核 8 线程的 x86 处理器 (64位)
  • 至少 16 GB 内存
  • 至少 64 GB 空闲磁盘空间 (固态硬盘)
  • 至少 1920x1080 的显示器
  • 显卡 NVIDIA RTX2070 或更高
  • 至少 6 GB 显存
  • NVIDIA 驱动版本 510.54 及以上
  • CUDA 版本 11.5 及以上 [下载 CUDA]

最低配置

(这是能够正常运行 Zeno 的最低要求配置)

  • Windows 10 及以上
  • ... 或 Ubuntu 18.04 及以上
  • x86 处理器 (64位)
  • 至少 4 GB 内存
  • 至少 4 GB 空闲磁盘空间
  • 至少 1920x1080 的显示器
  • 任意支持 OpenGL 3.2 的显卡
  • ... 或没有显卡（会自动切换为 CPU 软光栅渲染）
  • 提示：没有英伟达显卡，则应选择 cpu 版本的安装包下载。
  • cpu 版具有完全相同节点，只不过会在 cpu 上相对更慢地运行（得益于 zpc 的多后端技术）。
  • cpu 版没有 OptiX 实时光线追踪渲染器，但仍具有通用的 OpenGL 渲染器。
